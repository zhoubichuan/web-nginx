# 权限问题

## /etc/sudoers

```sh
# 定义一个命令别名
Cmnd_Alias SYSTEMCTL = systemctl start nginx, systemctl stop nginx, systemctl restart nginx
# 允许用户 'nginx' 以 root 身份执行上述命令，且无需输入密码
nginx ALL=(root) NOPASSWD: SYSTEMCTL
```

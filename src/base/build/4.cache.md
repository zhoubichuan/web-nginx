# 四.配置缓存

::: tip 前言
两种方式：1.强制缓存；2.协商缓存
:::

## 1.校验本地缓存是否过期

| 类型          | 种类                           |
| ------------- | ------------------------------ |
| 检验是否过期  | Expires,Cache-Control(max-age) |
| Etag          | Etag                           |
| Last-Modified | Last-Modified                  |

## 2.expires

添加 Cache-Control、Expires 头

| 类型   | 种类                 |
| ------ | -------------------- |
| 语法   | expires time         |
| 默认   | expires off          |
| 上下文 | http,server,location |

```bash
location ~ .*\.(jpg|png|gif)$ {
        expires 24h;
}
```

# 操作

## 1.重启

::: demo

```vue
<template>
  <div>
    <web-button type="success" @click="run('sudo systemctl restart nginx.service')">重启</web-button>
    <pre style="background: wheat;"> {{res}}</pre>
  </div>
</template>
<script>
export default {
  data() {
    return {
      res:''
    }
  },
  methods: {
    async run(task) {
      this.res  = await this.$api?.example.query(task)
    },
  },
mounted(){
    document.addEventListener("back",()=>{
        alert("=＂)
    })
}
}
</script>
```

:::

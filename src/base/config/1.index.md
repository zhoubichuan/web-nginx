# nginx 操作

## 1.重启

::: demo

```vue
<template>
  <div>
    <web-button type="success" @click="run(`su - nginx -c "sudo systemctl restart nginx"`)">重启</web-button>
    <pre style="background: wheat;"> {{res}}</pre>
  </div>
</template>
<script>
export default {
  data() {
    return {
      res:''
    }
  },
  methods: {
    async run(task) {
      this.res  = await this.$api?.example.query(task)
    },
  }
}
</script>
```

:::

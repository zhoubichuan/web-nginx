# 五.跨域

前言

两种方式：1.设置响应头信息，浏览器读取接口响应头信息跨域；2.使用代理服务器的方式跨域

# 1.CORS 方式

类型 种类
语法 add_header name value
默认 add_header --

location ~ .*\.json$ {
    add_header Access-Control-Allow-Origin http://localhost:3000;
    add_header Access-Control-Allow-Methods GET,POST,PUT,DELETE,OPTIONS;
    root /data/json;
}
